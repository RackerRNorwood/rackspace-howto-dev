You can access Rackspace CDN in the following ways:  

 

  • You can access and use Rackspace CDN through the GUI interface of the Cloud Control Panel. Click Storage in the top navigation bar and select CDN, as shown in the following image.

  • You can also access Rackspace CDN under Networking in the top navigation bar.
  • You can interact with Rackspace CDN programmatically through the Rackspace CDN API. For instructions, see the API Developer Guide, located in docs.rackspace.com. Also, the SDKs developed for Rackspace CDN are at Rackspace CDN SDKs.
