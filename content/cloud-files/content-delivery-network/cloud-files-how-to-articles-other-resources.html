SDKs

The Rackspace Cloud Files API is a RESTful API, with support for all major bindings listed below:

  • Java SDK
  • .NET SDK
  • PHP SDK
  • Python SDK
  • Ruby SDK
  • node.js SDK
  • Go SDK
