You can use Rackspace Intelligence to help you observe the status of your database if you set up the database to be visible to Rackspace Intelligence. You can do this through the Cloud Control Panel by following these steps:

  1. Create a database. For instructions on creating a database at the command-line by using the trove client, see Create Your First Cloud Database.
     
  2. Establish monitoring checks and alarms for the database as described in Monitoring Cloud Databases in the Cloud Control Panel.
